# AI-Powered Predictive Maintenance using Digital Twin

This project is an AI-based predictive maintenance system developed during a hackathon.

The system uses an Autoencoder model for anomaly detection and machine health prediction using parameters like temperature, load, and vibration.

## Technologies Used
- Python
- Machine Learning
- Autoencoder
- Digital Twin
- HTML Dashboard
- Flask Backend

## Features
- Anomaly detection
- Health score prediction
- Remaining Useful Life (RUL)
- Real-time monitoring dashboard
- What-if simulation
